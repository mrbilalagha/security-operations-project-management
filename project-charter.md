# Project Charter

## Project Objective
Deploy and stabilize security monitoring and patch management processes across multiple client environments while minimizing service disruption.

## Scope
In Scope:
- Endpoint security deployment coordination
- Patch scheduling and tracking
- Basic monitoring setup
- Documentation and handover

Out of Scope:
- Security architecture redesign
- Custom software development

## Constraints
- Limited technical resources
- Remote teams across time zones
- Fixed delivery timelines

## Success Criteria
- Security tools deployed as planned
- No major service outages
- Clear documentation delivered
